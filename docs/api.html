Contact Form API

Submit contact form data to our backend service.

Endpoint

POST /api/contact

Request Body

{
  "name": "John Doe",
  "email": "john@example.com",
  "subject": "Inquiry",
  "message": "Hello, I have a question..."
}

Response

{
  "success": true,
  "message": "Message sent successfully"
}

Newsletter API

Subscribe users to our newsletter.

Endpoint

POST /api/newsletter

Request Body

{
  "email": "user@example.com"
}

Response

{
  "success": true,
  "message": "Successfully subscribed to newsletter"
}
